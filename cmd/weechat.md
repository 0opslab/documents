# title{weechat - 一款快速的轻量级 IRC 客户端}

```bash
# Set unread marker on all windows
#在所有窗口上设置未读标记
Ctrl-s Ctrl-u

# Switch buffer left
#切换缓冲区了
Ctrl-p, Alt-left
# Switch buffer right
#切换缓冲区吧
Ctrl-n, Alt-right
# Next buffer with activity
#下一个缓冲区有活动
Alt-a
# Switch buffers
#切换缓冲区
Alt-0...9

# Scroll buffer title
#滚动缓冲区标题
F9/F10
# Scroll nick list
#滚动昵称列表
F11/F12
```
